# sample-api

Sample api that exposes GET, POST and PUT endpoints for updating clients and jobs.

### Client Endpoints Input Formats

#### Create/Post
```
{
  "cName": "exampleName"
}
```
#### Read/Get
```
{
  "id" : "exampleId"
}
```

#### Update/Put
```
{
  "id" : "exampleId",
  "cName": "exampleName"
}
```
#### Delete
Not Implemented
